# Memory Matching Game (jQuery)

## Project Overview
This project is a Memory Matching Game built using HTML, CSS, JavaScript, and jQuery. The objective of the game is to flip cards and find matching pairs.

The game demonstrates DOM manipulation, event handling, and game logic implementation using jQuery.

## Features
- Dynamic card generation
- Random card shuffle
- Card flip interaction
- Match detection
- Move counter
- Timer
- Sound effects for match and wrong attempts
- Win message when all pairs are matched

## Technologies Used
HTML  
CSS  
JavaScript  
jQuery  

## How to Play
1. Click any card to flip it.
2. Flip two cards to check if they match.
3. If the cards match, they remain open.
4. If the cards do not match, they flip back after 1 second.
5. Continue until all pairs are matched.
